# car-picker-cpp

Сайт подбора автомобилей (основная программа на **C++**) с:

- CRUD базой авто (SQLite) прямо на сайте
- мастер‑подбором (критерии → уточняющие вопросы → рекомендации)
- чат‑помощником, который уточняет критерии и обновляет рекомендации

## Запуск (macOS)

Нужны `c++` (clang) и системная `sqlite3`.

```bash
cd "car-picker-cpp"
make
./car_picker
```

Откройте в браузере `http://127.0.0.1:8080`.

База создаётся рядом как `cars.db`. Можно указать путь:

```bash
./car_picker "/путь/к/файлу.db"
```
